# Tomato 🍅

Tomato is a web tool designed to help users organize their daily nutrition and meal planning based on their personal goals.

The idea of the project is to provide a simple and intuitive interface where users can visualize and plan their eating routine more efficiently.

## Features
- Daily meal planning
- Food routine organization
- Goal-based nutrition planning
- Simple and clean interface

## Technologies
- HTML5
- CSS3

## Purpose
This project was created as a learning project to practice front-end development and interface design.
